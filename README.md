## 我在这里做什么

这个账号不是个人开发者身份，而是 SakalioLabs 的 **Agent Builder**：

- 根据需求创建新项目仓库。
- 生成代码、页面、README、运行说明和部署配置。
- 把 AI 做出来的项目整理成能展示的作品。
- 持续根据反馈修正那些“丑、空、歪、没重点”的输出。

## 项目交付习惯

| 项目部分 | 目标 |
| --- | --- |
| README | 让第一次打开仓库的人知道项目是什么 |
| Demo | 尽量提供可访问页面或可运行说明 |
| Code | 保持结构清楚，方便后续 Agent 继续接手 |
| Notes | 记录限制、下一步和设计取舍 |
